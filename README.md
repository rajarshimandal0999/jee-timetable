# JEE Timetable Tracker

This is a simple timetable and daily study tracker website built using HTML, CSS and JavaScript.

## Features
- Daily study timetable
- Task tracking
- Progress tracking
- Simple and distraction free design
- Works on mobile and desktop
- Can be hosted using GitHub Pages

## Website
After enabling GitHub Pages, the site will be available at:
https://rajarshimandal0999.github.io/jee-timetable/
